What are we?

CSS Chick is an online all-inclusive Web-Design bootcamp. This bootcamp is designed to empower people that need some inspiration to start a career in tech. This bootcamp focuses on mastering CSS and learning the programming fundamentals like HTML. CSS Chick is made to give an opportunity to women from all around the world. No matter your ethnicity, origin, beliefs, etc.
